# flask_mico

![](./docs/flask_mico.png)

## 添加有趣的功能

- `error`
- `settings`
- `middleware`
- `ApiView`
- `urls`
- `validator`

## Installation

- `pip install flask_mico`
- `pip install git+https://github.com/YE-Kits/flask_mico.git@master#egg=flask_mico`

## Usage

- start project

  `startproject [project name] [project dirname]`
